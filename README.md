# 📺 UI About Like a TV

A retro TV-inspired UI built with **HTML**, **CSS**, and **Vanilla JavaScript**.

![demo](preview.gif)

This project recreates a nostalgic television interface where users can turn the TV on/off and switch displayed images using number buttons.

The project is continuously improved as part of my JavaScript learning journey.

## ✨ Features

### 🔌 Power ON / OFF

- Turn the TV screen on and off
- Change button appearance based on power state
- Manage power state using JavaScript

### 🔢 Channel Buttons (1–9)

- Switch images by pressing number buttons
- Buttons work only when the TV is powered on
- Images are loaded from GitHub Pages

### 🎨 Retro TV Design

- Vintage television-inspired interface
- Simple and intuitive user interaction

